日報表

日報表使用說明

  • 本日預估營收：依排課數量和預約狀態，填入當天營收目標
  • 本日實際營收：當日結算後填入實際營收（教練課+團課+自主訓練+保健品+場租）
  • 本日學員數：當日上課學員人次（含團課、一對一、一對二、自主訓練）
  • 達成率 / 客單價 / 月營收累計 / 累計達成率：系統自動計算
  • 天氣：天氣影響中高齡學員出席率，記錄後可分析趨勢
  • 編輯方式：點擊任何「-」欄位即可輸入數字，按 Enter 儲存
月收支紀錄

收支表使用說明

  • 藍色欄位（直接成本）：每日進貨的原物料費用，如蛋、牛奶、麵包等
  • 紫色欄位（營業費用）：場館日常營運費用，如水電、清潔、耗材等
  • 營業額 / 其他收入：與日報表同步，填其中一邊即可
  • 日損益：= 營業額 + 其他收入 - 當日所有支出（自動計算）
  • 新增/刪除欄位：點擊右上角「管理支出欄位」可自訂成本和管銷的分類項目
  • 編輯方式：點擊任何「-」欄位即可輸入金額，按 Enter 或點擊其他地方自動儲存
損益表

損益表使用說明

  • 營業收入 / 直接成本：自動從日報表和收支表彙總，無需手動填寫
  • 營業費用（日常）：自動從收支表的紫色欄位彙總
  • 固定費用：點擊「+ 新增固定費用」新增每月固定支出（場租、保險、記帳費、律師費等）
  • 編輯固定費用金額：點擊金額數字即可修改，按 Enter 儲存
  • 百分比：各項目佔營業總額的比例（自動計算）

月報總覽使用說明

  • 此頁面自動彙整所有月份的數據，方便對比不同月份的營業表現
  • 資料來源：每個月的日報表和收支表填寫越完整，統計越準確
  • 圖表會在累積 2 個月以上的資料後自動顯示月營收趨勢和達成率曲線
  • 匯入資料：可將其他事業體匯出的 JSON 檔匯入，資料會自動累積
